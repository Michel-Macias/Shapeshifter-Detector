# Identify-files

A Python-based cybersecurity tool to identify file types using magic numbers (file signatures).

## Description
This tool analyzes the binary signature (magic numbers) of files to determine their true type, independent of their file extension. This is crucial for detecting obfuscated malicious files or recovering files with missing extensions.

## Features
- Database of common file signatures (Magic Numbers).
- CLI for easy usage.
- Support for single file or directory scanning.
